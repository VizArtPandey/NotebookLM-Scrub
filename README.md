# NotebookLM Scrub

The ultimate tool for scrubbing NotebookLM branding and converting slides into editable PowerPoint presentations. 

## Features
- **Zero Data Capture**: All processing is done locally in your browser. No files are ever uploaded.
- **Insta-Scrub**: Remove branding from PDF slides in seconds using a heuristic "healing" engine.
- **Editable PPTX**: Convert your slides into PowerPoint decks with editable text and image layers.
- **Privacy First**: Built with transparent, open-source technology.

## Deployment
This is a static React application. You can easily deploy it to Vercel:
1. Create a new GitHub repository and push this code.
2. Sign in to [Vercel](https://vercel.com).
3. Import the repository and click **Deploy**.
